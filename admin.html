Cette liste sert de secours quand aucun client n'est sélectionné dans le bon de livraison, et de point de départ lors de la création d'un nouveau client. Faites glisser ⠿ pour changer l'ordre.

Export / Import de configuration

Sauvegardez votre configuration (modèle par défaut, colonnes, champs, en-tête) et la liste de vos clients dans un fichier JSON, ou restaurez-en une depuis un fichier existant.

Exporter la configuration

Téléchargez un fichier apajh_config.json contenant tous vos réglages (modèle par défaut, colonnes, champs, en-tête) ainsi que tous vos clients et leurs modèles d'articles respectifs.

Importer une configuration

L'import remplace le modèle par défaut, les colonnes, les champs et l'en-tête. Pour les clients, une confirmation séparée vous sera demandée et les clients existants du même nom seront mis à jour plutôt que dupliqués.

Réinitialisation

Remettez la configuration d'usine. Toutes vos modifications seront perdues.

Réinitialiser la configuration

Cette action est irréversible et remettra tous les articles, colonnes et champs aux valeurs par défaut.
